## EyeChake

An eye care system that includes all aspects of an eye care clinic.

#### Installation
1. Clone the project and run the following commands
- <code>cd /path/to/project</code>
- <code>composer install</code>
- <code>npm install</code>
- <code>php artisan migrate</code>
- <code>php artisan db:seed</code>
2. Run the project with the commands
- <code>cd /path/to/project</code>
- <code>php artisan serve</code>
- <code>npm run dev</code> (This is not necessary if the files are already built using <code>npm run build</code>)
